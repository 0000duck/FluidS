## Download ##

Download available at <https://github.com/zigzagbob/FluidS/releases>.

For game body, please download at <http://pan.baidu.com/s/1gdlnUMZ> (key: vk8i | decompress pass: 7yzone).

Running game body may require DirectX, shortcut <https://download.microsoft.com/download/1/7/1/1718CCC4-6315-4D8E-9543-8E28A4E18C4C/dxwebsetup.exe>.

## Usage ##

 1. Download and decompress the package of game body.
 2. Download the package of patch.
 3. Decompress the patch to game path.
 4. Double click `SL5.exe`.

## Build ##

 1. Clone the repo using Git.
 2. Install Microsoft Visual Studio 2013 + Qt 5.4.0.
 3. Open `SL5.sln` as admin.
 4. Perform Build action.
